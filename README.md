# 🎂 Birthday Capsule

A simple Java command-line program that lets you write a message, goals, and a motivational quote to your future self — a personal "time capsule" you can look back on next birthday.

## Overview

Birthday Capsule prompts you for:
- A message to your future self
- Your 3 biggest goals for the year
- A motivational quote that fuels you

It then saves everything into a text file, so you can revisit it on your next birthday and see how far you've come.

## Repository Structure
Birthday-capsule/
├── birthdaycapsule2025.java     # Main Java program
├── out/production/Bithdaycapsule25/   # Compiled build output
└── .idea/                        # IntelliJ IDEA project files

## Requirements

- Java JDK 8 or above

## Usage

1. **Compile the program**
```bash
   javac birthdaycapsule2025.java
```

2. **Run it**
```bash
   java birthdaycapsule2025
```

3. **Follow the prompts**
   - Enter a message to your future self
   - Enter your 3 biggest goals for the year
   - Enter a motivational quote

4. **Check your capsule**
   Your responses are saved to `birthdaycapsule25.txt` in the same directory, ready to be opened and reflected on next year.
